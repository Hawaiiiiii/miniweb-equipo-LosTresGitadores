Equipo LosTresGitadores

Proyecto colaborativo usando Git y GitHub

Fecha de creación: 1 de noviembre de 2025

Sobre este proyecto

Este es un proyecto colaborativo desarrollado como parte de la actividad "Mini web del equipo – Colaboración real con GitHub" del curso de 2º DAW.

El objetivo es practicar el trabajo en equipo utilizando Git y GitHub, aplicando conceptos de ramas, merges, pull requests y resolución de conflictos.
